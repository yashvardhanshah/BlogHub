Bloghub Documentation
Bloghub is a blog sharing platform built with the MERN stack, where users can create, read, update, and delete blogs, as well as interact with posts from other users.

üìÇ Project Setup
1. Clone the repository:
git clone https://github.com/Sanuj14/Bloghub.git
cd Bloghub
2. Install dependencies:
cd server
npm install

cd client
npm install
3. Create a .env file in the backend with the following:
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
4. Run the project:
cd server
npm start

cd client
npm start
